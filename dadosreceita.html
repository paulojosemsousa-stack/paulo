```html CNPJ Consulta

Consulta CNPJ - ReceitaWS

Faça o upload do seu arquivo CSV contendo uma coluna chamada 'CNPJ'. A ferramenta irá consultar cada CNPJ na API da ReceitaWS, respeitando um limite de 3 requisições por minuto, e gerará um novo arquivo CSV com as informações de DATA_ABERTURA e SÓCIOS.

Aguardando arquivo...
Download Resultado CSV

Como Usar:

  1. Prepare um arquivo CSV com uma coluna intitulada `CNPJ`.
  2. Clique no botão "Procurar" para selecionar seu arquivo CSV.
  3. Clique em "Processar CNPJs" para iniciar a consulta.
  4. Aguarde o progresso. Devido ao limite de 3 requisições/minuto, o processamento de muitos CNPJs pode levar tempo.
  5. Após a conclusão, um link de download aparecerá para o arquivo CSV resultante.

Atenção sobre CORS:

Este script faz requisições diretas a uma API externa. No ambiente web, isso pode esbarrar em problemas de CORS (Cross-Origin Resource Sharing) caso o servidor da API não permita requisições de origens diferentes do seu domínio. No entanto, a API ReceitaWS geralmente permite requisições de navegadores. Se você enfrentar erros relacionados a CORS, a solução ideal seria usar um proxy reverso no servidor ou rodar o script em um ambiente de desenvolvimento sem as restrições de CORS (o que não é recomendado para produção).
```
